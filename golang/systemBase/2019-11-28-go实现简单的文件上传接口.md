# 介绍和初始化项目

## 初始工作区
首先，我们需要增加一个工作区（GOPATH）路径用于我们的项目。

将你新的工作区加入到/etc/profile中的GOPATH环境变量中， 并在新工作区中，建立bin、pkg、src三个目录。

在src目录下创建FILESTORE-SERVER目录，初始的目录结构：

$GOPATH
├── bin
├── pkg
└── src
    └── FILESTORE-SERVER
	
## 初始化项目目录	
FILESTORE-SERVER/
├── handler
├── static
├── upload
├── main.go

- handler 业务逻辑处理
- static 存储静态的html文件
- upload 存储上传的文件
